# Header

## Header Options


#### Top Bar

*  Enable / Disable the Top Bar.
*  Change the Top Bar’s background color.
*  Change the Top Bar’s text color.

#### Main Header

*  Enable / Disable the Sticky Header.
*  Update the phone Number for support.
*  Update the Email Address for support.
